TRACKER
v1.0.0 · GPS Activity Tracker
A progressive web app for recording hiking and outdoor activities with real-time GPS tracking, route visualization, and activity history.

Works offline after first load. Install to home screen for the full native experience on iOS and Android.

Built with Leaflet.js + OpenStreetMap.
